CRITICAL — Unit 6 Motor Control (0% covered, asked every year)

Both DC motor speed control and 3-phase IM speed control are asked in Part C (15 marks each) every year. These are Part C questions = 30 marks potential. This is your biggest risk area.

Action: Search YouTube — "DC motor speed control chopper" and "3 phase induction motor v/f control" — pick any clear 15-20 min video. Or use Notes directly (these are conceptual topics, mostly block diagrams + explanations, not heavy numericals).

Topics That Look Easy But Are Tricky in Exam
TopicCommon MistakeWhat to Remember
Step-Up Chopper numericalUsing Vo = α·Vs (step-down formula) instead of Vo = Vs/(1-α)Step-UP: Vo = Vs/(1-α). Step-DOWN: Vo = α·Vs. α = ton/T
SCR latching vs holding currentConfusing the two — ILatching > IHolding alwaysLatching = min current to TURN ON. Holding = min current to STAY ON. ILatching > IHolding
IGBT latch-upNot explaining how to avoid itCaused by parasitic thyristor structure. Avoid: limit anode current, use emitter shorts, reduce dv/dt
Half-bridge inverter output voltageUsing full Vdc instead of Vdc/2Half-bridge: Vo(peak) = Vdc/2. Full-bridge: Vo(peak) = Vdc
3-phase converter Vdc formulaUsing 1-phase formula for 3-phase3-phase half-wave: Vdc = (3√3·Vm/2π)cosα. 6-pulse full: Vdc = (3√3·Vm/π)cosα
Buck-boost Vo signForgetting the negative signBuck-boost: Vo = −Vs·α/(1−α). Output is inverted. |Vo| = Vs·α/(1−α)
Half-wave rectifier formulasConfusing Vavg with VRMSHalf-wave: Vavg = (Vm/2π)(1+cosα), VRMS = (Vm/2)√(1 - α/π + sin2α/2π)
VSI 3-phase 120° vs 180° modeDrawing wrong firing sequence120° mode: each thyristor conducts for 120°. 180° mode: each for 180°. Firing sequence: 1-2-3-4-5-6 (30° apart)
Key Formulas You Must Have On Your Fingertips
CONVERTERS
1-ph half-wave: Vdc = (Vm/2π)(1+cosα)
1-ph full converter: Vdc = (2Vm/π)cosα
3-ph half-wave: Vdc = (3√3·Vm/2π)cosα
3-ph full-wave: Vdc = (3√3·Vm/π)cosα
RMS (half-wave R): Vrms = (Vm/2)√(1-α/π+sin2α/2π)
Rectification eff η = Pdc/Pac
Form factor FF = Vrms/Vavg
Ripple factor RF = √(FF²-1)
CHOPPERS + DC-DC
Step-down: Vo = α·Vs (α = ton/T)
Step-up: Vo = Vs/(1-α)
Buck: Vo = α·Vs
Boost: Vo = Vs/(1-α)
Buck-boost: Vo = -α·Vs/(1-α)
Duty cycle: α = ton/(ton+toff)
Chopping freq: f = 1/(ton+toff)
INVERTERS
Half-bridge Vrms = Vdc/(2√2)
Full-bridge Vrms = Vdc/√2
Fundamental V1 = (4Vdc/π)·(1/√2) [rms]
3-ph 120°: VL(rms) = 0.8165·Vdc
V/f = constant (IM freq control)
SCR KEY VALUES
Two transistor: Ia = (α2·Ig)/(1-α1-α2)
At turn-on: α1+α2 = 1
ILatching > IHolding
dv/dt protection: RC snubber
di/dt protection: series inductor
I²t = (Isurge)² · t (thermal rating)
DC motor: N ∝ (Va-IaRa)/φ
